Usage
===========

The API is kept as simple as the Toast API:

Create a SimpleToast for any CharSequence:

	SimpleToast.ok(Context, CharSequence);
    SimpleToast.error(Context, CharSequence);
    SimpleToast.info(Context, CharSequence);
